# Men of Stone

A static first version of the Men of Stone website, dedicated to Phil Stone.

Open `index.html` in a browser to view it locally. The current contact form is front-end only and can be connected to email, Google Sheets, or a database later.

## Assets

- `assets/photos/` contains local copies of selected Phil Stone images from the supplied folder.
- `assets/concept/men-of-stone-concept.png` is the generated visual reference used for the page direction.
